Whit this command you can sync your work directory with the browser

    $ browser-sync start --server --directory --files "*"

  • browser-sync: node module to sync your folder
  • start: init the module
  • --server: the sync module start as a server in background
  • --directory: Shows in the browser the directories that are synchronized
  • --files "*": specify that all files should be synchronized
